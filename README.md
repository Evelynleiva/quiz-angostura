# Quiz Angostura

Proyecto de aplicación web de preguntas y respuestas (quiz) desarrollado como parte de la asignatura de Proyecto Integrado
Permite gestionar preguntas, registrar respuestas de los usuarios y mostrar un ranking de resultados.

## Tecnologías utilizadas

- Python y Django para el backend.
- JavaScript / React y Vite para el frontend.
- HTML y CSS para la interfaz de usuario.
- Base de datos relacional MySQL 

## Objetivo del proyecto

Implementar un sistema de quiz que permita:
- Administrar preguntas y categorías.
- Registrar la participación de los usuarios.
- Visualizar puntajes y ranking de resultados.
- Compartir participacion en redes sociales
